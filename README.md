# API Tests for secby.ru

## Описание
Автотесты для проверки API сайта secby.ru в рамках Практической работы 8 курса "Введение в профессию".

## Что тестируется
Авторизация (получение токена при валидных/невалидных данных)  
Передача токена в заголовках запросов  
Получение профиля текущего пользователя  
Получение профиля по ID  
Проверка роли администратора  
Получение списка пользователей (пагинация)  
Проверка роли модератора  
Обновление профиля  
Доступ без авторизации (401/403)

## Технологический стек
- **Язык:** Python 3.12.5
- **Фреймворк:** pytest 7.4.3
- **Библиотеки:** requests, allure-pytest
- **API:** REST (secby.ru)
